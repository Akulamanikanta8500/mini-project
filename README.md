A premium, automated solution for **Sri Venkateswara University** to manage and generate Transfer Certificates (TC). This application streamlines the workflow from student application to digital certificate issuance with a modern, glassmorphism interface.

## ✨ Key Features

- **🚀 Student Portal**: Seamless application process with document upload capabilities.
- **🛡️ Admin Dashboard**: Secure administrative controls for reviewing and approving applications.
- **📜 Smart TC Generation**: One-click professional TC generation with:
  - Official SV University Branding
  - Dynamic Watermarking
  - High-Verification Fields
  - Print-Ready Layout
- **🎨 Glassmorphism UI**: Beautiful, semi-transparent design inspired by modern aesthetics.

## 🛠️ Technology Stack

- **Frontend**: [React.js](https://reactjs.org/), [Vite](https://vitejs.dev/), [React Bootstrap](https://react-bootstrap.github.io/)
- **Backend**: [Node.js](https://nodejs.org/), [Express.js](https://expressjs.com/)
- **Database**: [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)
- **Styling**: Vanilla CSS with Glassmorphism effects

## 🏁 Getting Started

### 1. Clone & Install
```bash
git clone https://github.com/Akulamanikanta8500/Tc-Application.git
cd Tc-Application
npm install
cd backend && npm install
cd ..
```

### 2. Environment Setup
Create a `.env` file in the `backend/` directory:
```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### 3. Run the Project
**Launch Backend:**
```bash
npm run server
```

**Launch Frontend:**
```bash
npm run dev
```
